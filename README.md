#おとりっぷ

##概要
自分に合ったイヤホンを質問形式で見つけれるWebサイト

##特徴
-質問に答えるだけでイヤホンが探せる
-Amazonのリンクなども貼ってるのですぐに購入ができる
-その商品が気になるときは商品Urlも貼っているのですぐに商品仕様が確認できる

##使用技術
-HTML
-CSS
-JavaScript
